    复古电脑博物馆

    　　复古电脑博物馆是一个在线交互式的计算机博物馆，在网页中重现电脑发展史。通过 JavaScript 模拟器来体验 1970 年代 —— 1990 年代流行的电脑硬件和软件。时空流转，沧海桑田，物是人非，但那些程序代码依然在跳动着。

    家用游戏博物馆

    　　一个在线交互式的电子游戏博物馆，通过 JavaScript 模拟器复活那些历史上由中国大陆及港澳台地区开发的，以家用游戏机和掌上游戏机为平台的电子游戏。用户无需安装通过浏览器即可体验到童年的乐趣。

    DOS游戏博物馆

    　　收录了 1898 款 DOS 游戏，这些古老的 DOS 游戏通过 JavaScript 模拟器在网页中重生，焕发了新的生机。用户无需安装即可体验。谨此向那个没有网游、没有手游、性能有限却又最有创造力的游戏黄金时代致敬！

    CPU博物馆

    　　一个致力于传承和推广 CPU 文化的在线博物馆，展览了多年来我收集的各种 CPU，从最早的 Intel 4004 到最新的 Intel Core，从 8 位到 64 位，从单核到多核，从英特尔到 AMD，从 x86 到 ARM，从桌面到移动，应有尽有。

    Flash档案馆

    　　Flash 曾经是互联网的一部分，也为互联网带来了繁荣。Flash 的落幕代表着一个时代的结束，它承载着我们许多的回忆。Flash 档案馆目前收录了 8000 多个历史上热门的各类 Falsh 动画，通过 Ruffle 技术进行渲染播放，无需安装插件。

    Apple Macintosh 软件专题

    　　从1984年到1997年，Apple Macintosh系列电脑，以其优秀的设计、易用的界面、高质的硬件和软件而受到了教育、艺术、设计等领域的青睐。通过本专题展，您将与Macintosh相遇，在浏览器中感受那段缺失的岁月。

    Amiga 软件专题

    　　Amiga电脑作为1980年代中期最为强大的16位多媒体电脑，在西方电脑爱好者中具有极高的地位。采用摩托罗拉680x0系列处理器，具有高分辨率，快速图形响应，多媒体任务的特点，特别为游戏方面做了专门设计。

    DOS病毒博物馆

    　　收录了1980年代和1990年代在家用计算机上传播的恶意软件程序。一旦他们感染了系统，有时会显示您已被感染的动画或消息。通过使用DosBox仿真模拟器，可让您安全地体验几十年前的病毒感染。

    Windows 3.x 软件专题

    　　Windows 3.0/3.1/3.2等都是在建立在DOS上的操作环境，并不是一个真正的操作系统，但就其意义而言却非常重大，因为它们实现了PC机操作的图形化。在此专题中，可通过浏览器再次使用这些Windows 3.x下的经典软件。

    DOS 软件专题

    　　DOS，全称是Disk Operating System，是现在Windows操作系统的前身。相对于Windows系列，DOS较难学习，不易操作，却有着Windows最为欠缺的自由和开放。此专题可在线使用古老DOS软件，不妨来重温一下过去自由的好时光吧！

    电脑学习机

    　　电脑学习机是我国计算机发展的一个重要分支，它是在特殊历史背景下的产物，让无数青少年对计算机的知识有了初步的了解。我们用电脑学习机玩游戏、打字、学编程等，留下了一段青春中最美好的记忆。

    史蒂夫的旧电脑博物馆

    　　早期的个人电脑与现在的电脑完全不同——它们很有个性！每种品牌都彼此不同，比以前的产品更令人兴奋，具有新的特性和功能。史蒂夫的旧电脑博物馆收录了 150 台旧电脑，致力于保存和展示这些老式计算机系统。
